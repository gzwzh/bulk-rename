# 鲲穹AI批量重命名

功能对标 Bulk Rename Utility 的批量文件重命名工具，使用 Python + PyQt6 开发。

## 功能特性

- **正则表达式替换** - 使用正则表达式进行高级匹配和替换
- **文本替换** - 简单的查找替换功能
- **字符移除** - 移除指定位置或特定字符
- **前缀/后缀添加** - 在文件名前后添加文本
- **自动编号** - 为文件添加序号
- **大小写转换** - 转换文件名大小写
- **自动日期** - 添加文件创建/修改日期
- **扩展名处理** - 修改文件扩展名
- **实时预览** - 即时查看重命名效果
- **拖放支持** - 直接拖放文件或文件夹
- **配置保存/加载** - 保存和复用重命名规则

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 使用说明

1. 通过左侧文件夹树选择目录，或直接拖放文件/文件夹
2. 在下方规则面板中勾选并配置需要的重命名规则
3. 实时预览新文件名（蓝色显示变化）
4. 点击"重命名"按钮执行操作

## 快捷键

- `F2` - 执行重命名
- `F5` - 刷新预览
- `Ctrl+O` - 打开文件夹
