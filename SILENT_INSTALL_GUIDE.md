# 静默安装指南

## 概述
批量重命名工具现在支持静默安装功能，可以在无用户交互的情况下自动完成安装，并自动创建桌面快捷方式。

## 静默安装命令

### 基本静默安装
```cmd
批量重命名_安装程序.exe /SILENT
```

### 完全静默安装（不显示任何界面）
```cmd
批量重命名_安装程序.exe /VERYSILENT
```

### 静默安装 + 不重启
```cmd
批量重命名_安装程序.exe /SILENT /NORESTART
```

### 指定安装目录的静默安装
```cmd
批量重命名_安装程序.exe /SILENT /DIR="C:\MyPrograms\批量重命名"
```

## 命令行参数说明

| 参数 | 说明 |
|------|------|
| `/SILENT` | 静默安装，显示进度条但不需要用户交互 |
| `/VERYSILENT` | 完全静默安装，不显示任何界面 |
| `/NORESTART` | 安装完成后不自动重启系统 |
| `/DIR="路径"` | 指定安装目录 |
| `/GROUP="组名"` | 指定开始菜单组名 |
| `/NOICONS` | 不创建开始菜单图标 |
| `/TASKS="任务列表"` | 指定要执行的任务 |

## 静默安装特性

### 自动创建桌面快捷方式
- 静默安装时会自动在桌面创建快捷方式
- 无需用户手动选择

### 默认安装位置
- 程序文件夹: `%ProgramFiles%\批量重命名\`
- 开始菜单: `批量重命名`

### 安装日志
- 静默安装过程会记录到系统日志
- 可通过事件查看器查看安装状态

## 使用示例

### 1. 使用提供的批处理文件
```cmd
silent_install.bat
```

### 2. PowerShell 脚本示例
```powershell
# 检查安装程序是否存在
if (Test-Path "dist\批量重命名_安装程序.exe") {
    Write-Host "开始静默安装..."
    Start-Process "dist\批量重命名_安装程序.exe" -ArgumentList "/SILENT", "/NORESTART" -Wait
    Write-Host "安装完成！"
} else {
    Write-Host "找不到安装程序文件"
}
```

### 3. 企业部署示例
```cmd
REM 网络部署脚本
\\server\share\批量重命名_安装程序.exe /VERYSILENT /NORESTART /DIR="%ProgramFiles%\批量重命名"
```

## 卸载

### 静默卸载
```cmd
"%ProgramFiles%\批量重命名\unins000.exe" /SILENT
```

### 完全静默卸载
```cmd
"%ProgramFiles%\批量重命名\unins000.exe" /VERYSILENT
```

## 注意事项

1. **管理员权限**: 某些情况下可能需要管理员权限
2. **防病毒软件**: 确保防病毒软件不会阻止安装
3. **网络环境**: 在企业网络环境中部署时，确保有足够的权限
4. **安装验证**: 安装后可检查程序是否正确安装到指定位置

## 故障排除

### 常见错误代码
- `0`: 安装成功
- `1`: 一般错误
- `2`: 用户取消安装
- `3`: 致命错误
- `4`: 目录不存在
- `5`: 访问被拒绝

### 检查安装状态
```cmd
REM 检查程序是否已安装
if exist "%ProgramFiles%\批量重命名\批量重命名.exe" (
    echo 程序已成功安装
) else (
    echo 程序未安装或安装失败
)
```
